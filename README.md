# Challenge Alura Store: Analisis de Datos Tiendas de Don Juan

Este repositorio contiene el proyecto final practico del Challenge Alura Store, desarrollado como parte de la Especializacion en Data Science de ONE (Oracle Next Education). 

El objetivo principal de este proyecto es aplicar tecnicas de limpieza, manipulacion y visualizacion de datos utilizando Python para evaluar el rendimiento comercial y logistico de cuatro sucursales (Tiendas 1 a 4).

## Objetivos del Analisis

A traves del procesamiento de las bases de datos de las tiendas, el analisis responde a preguntas clave del negocio divididas en 5 metricas fundamentales:

1. Analisis de Facturacion: Comparativa de los ingresos totales generados por cada sucursal.
2. Top Productos: Identificacion de los productos estrella (mas vendidos) y el inventario estancado (menos vendidos).
3. Analisis de Categorias: Evaluacion del volumen de ventas agrupado por categoria de producto.
4. Satisfaccion del Cliente: Promedio de calificaciones otorgadas por los compradores en cada tienda.
5. Costos Logisticos: Calculo del costo de envio promedio para detectar oportunidades de optimizacion.

## Tecnologias Utilizadas

* Python
* Pandas
* Matplotlib
* Jupyter Notebook

## Estructura del Proyecto

* base-de-datos-challenge1-latam: El cuaderno de Jupyter con todo el codigo fuente, funciones de calculo y generacion de graficos.
* Informe_Don_Juan.pdf: El reporte ejecutivo final que traduce los hallazgos tecnicos en decisiones de negocio.
* datasets/: Carpeta que contiene los archivos CSV originales de las 4 tiendas analizadas.
